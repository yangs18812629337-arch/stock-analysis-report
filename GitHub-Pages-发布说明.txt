GitHub Pages 发布方式

当前 GitHub 仓库：
https://github.com/Young9337-arch/stock-analysis-report

GitHub Pages 公网页面：
https://Young9337-arch.github.io/stock-analysis-report/

我会上传这些文件：
1. index.html
2. daily-market-report.html
3. .nojekyll

如果需要重新开启 GitHub Pages，在 GitHub 仓库里打开：
Settings -> Pages

Source 选择：
Deploy from a branch

Branch 选择：
main / root

保存后，GitHub 会生成一个公网地址：
https://Young9337-arch.github.io/stock-analysis-report/

以后别人手机、电脑都应该打开这个 https 地址，不要再打开 localhost。
